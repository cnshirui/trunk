Platform Stack

The base Heroku platform stack is composed of widely deployed open source software:

  • Operating System: Debian 4.0
  • Ruby: MRI 1.8.6
  • Ruby App Server: Thin
  • Web Server: Nginx 0.6.32
  • Database: PostgreSQL 8.3.5
  • HTTP cache/accelerator: Varnish 2.0.2

See How it Works for more information about these components; the Platform Architecture section is especially relevant.

Ruby Frameworks and Libraries

While each application can install any number of custom gems, Heroku includes a large number of the most popular Ruby frameworks and libraries as part of the base platform. The following is a partial list of gems installed by default for all apps. See the following section for a comprehensive list.
